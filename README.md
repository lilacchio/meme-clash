Prediction markets today are **boring** and **gatekept**.
- **Centralized**: Only the platform decides what you can bet on.

- **Extractive**: Users provide all the liquidity and volume, but the platform keeps 100% of the fees.

## ‚ö° The Solution: MemeClash
**MemeClash** is a permissionless, high-energy prediction market built for the **Meme Economy**. We turn gamblers into hosts.

### Key Features
- **üî• Permissionless Creation**: Anyone can create a market instantly by burning **0.1 SOL** (Devnet). No gatekeepers.
- **üí∞ Creator Yield**: The unique "Kickstarter" mechanic ‚Äî **Creators keep 50% of the house fees** generated by their market. The more viral your market, the more you earn.
- **üé® Permissionless Chaos UI**: A Neo-Brutalist design system that embraces the raw energy of crypto twitter.

## üöÄ Getting Started

### Prerequisites
1.  Node.js 18+
2.  **Phantom Wallet** or Solflare (Browser Extension)
3.  **SOL (Devnet)**: You need Devnet SOL to create markets. Get it from [faucet.solana.com](https://faucet.solana.com/).

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/meme-clash.git

# Enter directory
cd meme-clash

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## üß™ How to Test (For Judges)

### 1. Connect Wallet (Devnet)
Ensure your Phantom wallet is set to **Developer Settings > Change Network > Devnet**.

### 2. Create a Market
1.  Click **`+ CREATE`** in the navbar.
2.  Fill in a question (e.g., "Will SOL flip ETH?").
3.  Click **"PAY 0.1 SOL & CREATE"**.
4.  Approve the transaction in your wallet.
    *   *Verification*: Check your wallet history to see the 0.1 SOL transfer.

### 3. Place a Bet
1.  Go to the specific market page.
2.  Select **YES** or **NO**.
3.  Click **"Buy"**. The Price Chart will update instantly based on CPMM logic.

### 4. Check Earnings
1.  Go to **`PROFILE`**.
2.  View your **"Claimable Earnings"**. As users bet on your market, your 50% cut accumulates here!
3.  Simulate a "Withdraw" to claim your rewards.
